# 🤖 PlacementPrep AI - Smart Placement Preparation Portal  

A modern and intelligent **AI-driven placement preparation platform** built using **React.js, JavaScript, and Tailwind CSS**, designed to help students excel in campus placements through mock interviews, resume analysis, skill-based tests, and personalized job recommendations.

## 💡 Key Highlights  

- 🎤 **Mock Interview System** — Webcam-based interview with simulated emotion tracking  
- 📄 **AI Resume Analysis** — Skill extraction, career path prediction, and personalized content  
- 🗣️ **Communication Practice** — Speech-to-text using Web Speech API  
- 🧠 **Skill-Based Mock Tests** — Timed MCQs with performance analytics  
- 💼 **AI Job Matching Portal** — Matches jobs based on user skills with percentage score  
- 📊 **Performance Tracking** — Saves user progress using localStorage  

---

## 🛠️ Tech Stack  

- **React.js** — Single Page Application (SPA)  
- **JavaScript (ES6+)** — Core logic and interactivity  
- **Tailwind CSS** — Modern responsive UI  
- **Browser APIs** —  
  - Web Speech API (voice recognition)  
  - getUserMedia API (camera access)  
  - File API (resume upload)  
  - localStorage API (data persistence)

## ⚙️ Features Breakdown  

### 🎤 Mock Interview  
- Opens webcam using getUserMedia API  
- Simulates facial expression detection  
- Provides timed HR questions  
- Tracks performance and saves results  

### 📄 Resume Analyzer  
- Upload resume using File API  
- Generates skills and career path  
- Identifies skill gaps  
- Provides personalized questions and resources  

### 🗣️ Communication Practice  
- Real-time speech-to-text conversion  
- Activities like reading, listening, and speaking  
- Score-based evaluation system  

### 🧠 Mock Tests  
- Multiple categories (DSA, DBMS, Web Dev, OOP)  
- Difficulty-based filtering  
- Timer-based tests  
- Instant results with analytics  

### 💼 Job Matching Portal  
- Matches jobs based on skills  
- Displays match percentage  
- Search and filter options  
- Tracks applied jobs

## 📊 Project Insights  

- 💻 ~3,500+ lines of code  
- ⚛️ Built as a Single Page Application (SPA)  
- 📦 Uses multiple structured datasets (skills, jobs, questions, courses)  
- 🔄 50+ state variables for dynamic UI  
- 🌐 Fully client-side application (no backend)  

---

## ⚠️ Note  

- Facial Expression Recognition is simulated (not real AI)  
- Resume parsing is rule-based and does not extract actual PDF content  
- No backend or database is implemented
